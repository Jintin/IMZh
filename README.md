# IMZh
IMZh is a localization tool to translate IMDB movie name to Chinese

## Installation
Visit [Download page](https://chrome.google.com/webstore/detail/imzh/plkfiilaiekcjgahedeodpkleeajhhhd) and click `ADD TO CHROME` button.
